# Dnd2Install

A drag and drop installer for Linux.

<img src="screenshot.png" alt="Screenshot" />

## Get started

1. Right click on file `.rpm`, `.deb`, `.tar.gz`, `.zip`
2. Click "Open with" > "Drag and drop to install"
